# Camisa100 ⚽🇧🇷

Jogo de adivinhação de craques do futebol, brasileiros e internacionais. A cada rodada, uma camisa sorteia um jogador e você precisa descobrir quem é usando o mínimo de dicas possível — quanto menos dicas, mais pontos!

## Como jogar

1. Escolha o nível de dificuldade: 😊 fácil, 😐 médio ou 😠 difícil.
2. Toque na camisa para sortear um jogador.
3. A primeira dica é grátis. As próximas custam pontos diferentes (posição, estado natal, época, clube marcante, título marcante).
4. Digite o nome do jogador e clique em **Chutar**.
5. Errou? Perde 5 pontos e pode tentar de novo. Acertou? Pontuação vai pro seu placar total.
6. Não sabe? É só clicar em **Revelar resposta**.

## Funcionalidades

- Mais de 200 jogadores, brasileiros e internacionais, com 3 níveis de dificuldade
- Sistema de pontuação com pistas que custam pontos diferentes
- Sons sintetizados via Web Audio API (sem arquivos externos)
- Botão para ativar/desativar som
- Totalmente responsivo, feito com HTML, CSS e JavaScript puro
