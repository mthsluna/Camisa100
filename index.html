    Quem é o jogador?

    Base de jogadores selecionada manualmente com atletas brasileiros (seleção e clubes), pois APIs públicas de futebol exigem chave secreta ou não liberam chamadas diretas do navegador (CORS), o que inviabilizaria rodar tudo dentro deste artefato. Alguns dados (época, apelido, naturalidade) são simplificados para fins de jogo.
